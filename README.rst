Darkseid
========

A python library to interact with comic archives.

Installation
------------

PyPi
~~~~

.. code:: bash

    $ pip3 install --user darkseid

GitHub
~~~~~~

Installing the latest version from Github:

.. code:: bash

    $ git clone https://github.com/Metron-Project/darkseid
    $ cd darkseid
    $ python setup.py install
